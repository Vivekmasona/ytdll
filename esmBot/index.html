esmBot

esmBot is an open source and entertainment-focused Discord bot made using Eris with soundboard, utility, and image manipulation commands, alongside many others.

Features

  • Use the image manipulation commands to warp, create, or drastically change images.
  • The tags system lets you save content that you can access at a later time.
  • Play a variety of sounds in voice chat with the soundboard commands.
  • Generate and read QR codes with the qrread and qrcreate commands.
  • Search for videos and images without leaving Discord.
  • Shorten and unshorten URLs on the fly.
  • And more!

FAQ

  • How do I use commands? What prefix do I use?
    • The default prefix on most servers is &. However, if you don't know the prefix currently being used, you can mention the bot instead of using the prefix (@esmBot help).
  • The resulting image is embedded in a frame, what does this mean?
    • The image generated by esmBot was larger than 8 MB, which is the Discord upload limit. To get around this, esmBot automatically uploads images larger than 8 MB to an external server. Keep in mind, however, that images uploaded this way expire within 5-10 minutes of being posted, so save your image before it does!
  • I got an error when running a command, what do I do?
    • That's not good! Report it to me via the GitHub issue tracker or on the esmBot Support server.
